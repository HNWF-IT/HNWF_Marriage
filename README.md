## About

This project aims to solve the issue of managing and finding best matches for those who recently converted to Islam, and are looking for marriage, Please do not copy, sell or re-distribute without permission.

- What is the goal of the project?: Aimed to find best matches for those converting to Islam 
- What technologies does it use?: Mongo, Express, React, Node [MERN-Stack]

## Installation

Follow these steps to install and run the project locally.

1. Clone the repository:

   ```bash
   git clone https://github.com/AhmadFaraz-125/HNWF_Marriage.git

2. Move to the client & server directory (in 2 different terminals):

   ```bash
   cd client
   cd server 
   
3. Install dependencies in both terminals:

   ```bash
   npm install
   
4. For development, run in both terminals:

   ```bash
   npm run dev
